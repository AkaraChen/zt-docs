# 何为诸天？

诸天之名，取于话本中「诸天万界」一词，盖因，无量网络之中，每个域名都如一方世界，无数个世界便组成了诸天。

## 因

许久前的一天，吾看见了一件名为 [开往-友链接力](https://github.com/volfclub/travellings) 的法宝，此法宝精妙至极，但是细想来，感到仍有不足，便想炼制一法宝，将所有的博客索引至其上呢？

## 果

历经无量劫，诸天，便诞生了。

诸天乃是一博客聚合平台，支持甚多博客系统与CMS，每日清晨由方外神物 Github Action 从无量网络中获取信息，在诸道友访问诸天之时，诸天将随机推送数篇玄妙文章，其性质，可称为「另类友链」
